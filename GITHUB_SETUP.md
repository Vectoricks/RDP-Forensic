## How to Add These to GitHub

### Option 1: Via GitHub Web Interface

1. Go to your repository: https://github.com/BetaHydri/RDP-Forensic
2. Click the **⚙️ Settings** gear icon (top right, near "About")
3. In the "About" section:
   - **Description**: Paste the repository description
   - **Website**: Add the URL
   - **Topics**: Add the topics one by one (they'll autocomplete)
   - Check ✅ **Include in the home page** if desired
4. Click **Save changes**

### Option 2: Via Repository Settings Page

1. Navigate to repository Settings
2. In the "General" section, find "Social preview"
3. Update description and website URL
4. Add topics in the repository home page "About" section
